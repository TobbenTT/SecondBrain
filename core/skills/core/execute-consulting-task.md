# Execute Consulting Task

## Purpose
Instrucciones de ejecucion para agentes consultores que generan documentos profesionales a partir de ideas y SOPs.

## Cuando Usar
- Cuando un agente consultor (staffing, training, finance, compliance, etc.) necesita generar un documento
- Cuando se ejecuta una tarea del pipeline Consulting

## Instrucciones

1. Analiza la idea/solicitud proporcionada.
2. Usando tu conocimiento de las Skills/SOPs, genera un OUTPUT ESTRUCTURADO y COMPLETO.
3. El output debe ser un documento profesional listo para presentar.
4. Usa formato Markdown con secciones claras, tablas donde aplique, y datos cuantitativos.
5. Se especifico: incluye numeros, plazos, responsables donde sea posible.
6. Incluye una seccion de "Proximos Pasos" al final.
7. Responde en espanol.

## Output Esperado
Documento en Markdown con:
- Titulo claro
- Secciones estructuradas con headers ##
- Tablas donde aplique (comparativas, cronogramas, presupuestos)
- Datos cuantitativos concretos
- Seccion final de "Proximos Pasos" con acciones asignables

## Restricciones
- No generar documentos genericos sin datos especificos
- No omitir la seccion de proximos pasos
- No mezclar idiomas (todo en espanol)
